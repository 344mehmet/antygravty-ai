               DOCKER TEMİZLİK KOMUTLARI
=============================================================

# Kullanılmayan tüm verileri temizle

# Sadece eski imajları temizle
docker image prune -a

# Sadece durmuş konteynerleri temizle
docker container prune

# Build cache temizle
docker builder prune

=============================================================
         DOCKER VHDX KONUMUNU DEĞİŞTİRME
=============================================================

Docker Desktop data'yı harici diske taşımak için:

1. Docker Desktop > Settings (Dişli İkonu)
2. Resources > Advanced
3. "Disk image location" yanındaki Browse butonuna tıkla
4. Harici diskte yeni klasör seç (örn: D:\Docker\data)
5. Apply & Restart
